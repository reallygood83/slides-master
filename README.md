# Paper2Slides - AI Presentation Generator for Obsidian

Transform your markdown notes into professional presentation slides using AI technology.

## Features

- **AI-Powered Generation**: Automatically convert markdown content into structured slides
- **Multiple Themes**: 5 professional themes (Academic, Doraemon, Minimalist, Corporate, Creative)
- **Flexible Resolutions**: 4K, FHD, HD, SD options
- **Export Formats**: HTML, PDF (planned), PPTX (planned)
- **Customizable Pipeline**: Fast or Normal mode with adjustable slide length
- **Progress Tracking**: Real-time progress modal during generation
- **Preview System**: Optional preview before final generation

## Installation

### BRAT (Beta Reviewers Auto-update Tester)

1. Install BRAT plugin in Obsidian
2. Add this repository: `reallygood83/Paper2Slides-Obsidian`
3. Enable the plugin in Settings

### Manual Installation

1. Download the latest release from GitHub
2. Extract to `.obsidian/plugins/paper2slides/`
3. Enable in Settings > Community Plugins

## Usage

1. Open any markdown note in Obsidian
2. Run command: "Generate slides from current note"
3. Configure options (theme, resolution, length)
4. Wait for AI processing
5. Find generated slides in `slides/` folder
